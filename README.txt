Для кооректной работы приложения необходимо выполнить следующие действия:

1) Скачайте папку из облачного хранилища по адресу: https://www.dropbox.com/sh/6mcf6g1zqbazc64/AABnn-ngkzY0FYznUY8H3UOJa?dl=0
2) Скопировать все файлы через FTP - клиент на сервер
3) Создать базу данных на сервере ( запомнить имя БД, пользователя БД и пароль от БД )
4) Открыть файл "connect_db.php" и изменить параметры соединения с БД ( для корректного изменения прописаны комеентарии )
5) Чтобы приложение работало максимально по своему функционалу, необходимо при первом запуске нажать кнопку "Добавить клиента"
и нажать кнопку "Добавить клиента". После добавления клиента приложение выдаст сообщение об успешном добавлении и вернет 
пользователя на главный экран. На этом установка для корректной работы приложения завершена.



Описание функционала приложения.

1) На главной странице присутствуют четыре кнопки:

	а) Добавить клиента;
	б) Список всех клиентов;
	в) Найти клиента;
	г) Просмотр карточки клиента.

а) В добавлении клиента необходимо указать обязательные поля для заполнения: "ФИО, дату рождения, пол и сотовый телефон". Поле
"Рабочий при наличии" не обязательно для заполнения.

б) Список клиентов представляет из себя таблицу со столбцами: ID, Фамилия, Имя, Отчество, День рождения, Пол, Телефон основной, Телефон доп.

в) Найти клиента можно двумя способами по фамилии - после чего необходимо выполнить ввод необходимой фамилии или по номеру телефона.
Если пользователь будет найден по одному из этих критериев, то отобразится таблица с данными по этому клиенту. 
P.S. В приложении предусмотрен поиск нескольких клиентов, например, если у них одинаковые фамилии.

г) Просмотр карточки клиента - все данные по клиенту. Выбрать клиента необходимо из выпадающего списка. Выпадающий список строится на 
основе таблицы клиентов. После выбора конкретного клиента надо нажать кнопку "Просмотреть" для вывода всех данных о клиенте. Вместе с данными
появляются две кнопки: "Удалить клиента" - при нажатии которой удалится выбранный клиент из базы и "Редактировать клиента". Редактирование 
клиента производится в новом окне, где все его данные заполнены в форме. Если необходимо удалить дополнительный номер телефона, нужно
просто стереть данные из формы. Изменить можно все поля, кроме пола, т.к. это уже будет не изменения клиента, а добавление нового.
После нажатия кнопки "Сохранить изменения" данные о клиенте обновятся.
P.S. В таблице необходимо наличие: времени создания клиента 
и времени обновления так, после редактирования поле "время_обновления" получит значения в виде даты и времени, которые до этого были пустыми.

2)На каждой странице кроме главной присутствует кнопка: "На главную", чтобы из любого места приложения всегда была возможность вернуться в "Главное меню".
